# gestion-des-heures-uvci
L'Université Virtuelle de Côte d'Ivoire (UVCI) repose sur un modèle pédagogique entièrement en ligne  basé sur la production et la mise à disposition de ressources numériques. Contrairement à un  enseignement présentiel classique, les activités des enseignants englobent la création, la structuration  et la mise à jour de contenus digitaux.
